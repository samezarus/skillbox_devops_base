

Добавление воторого источника кода к уже имеющемуся:

    Т.е. уже гит репа, и к ней нужно добавить ещё одну

    1. Копируем певую репу
        
        git clone <имя репы>

    2. Переходим в сколнипрваную репу:

        cd <имя репы>

    3. Добавляем второй источник(репу):

        git remote add <произвольное название источника> <url вротой репы>

    4. Проверка что вторая репа добавлена:

        git remote -v

    5. Получить коммиты второй репы:

        git fetch <ранее заданное название второго источника>

    6. Создать ветку для тестов второго источника:

        git branch <имя тестовой ветки> <название второго источника>/<ветка>

    7. Переключаемся на тестовую вутку:

        git checkout <имя тестовой ветки>

    8. Ещё раз пытаемся стянуть ветку из ворого источника для актуализации данных:

        git pull

    9. Закидываем код из второго источника внашу репу и делаем merge request:

        git push -u origin -o merge_request.create

    Пример:

        1. git clone mygitlab.ru/sameza/app.git
        2. cd ./app
        3. git remote add otto_app github.com/otto/app_xxx.git
        5. git fetch otto_app
        6. git branch otto_test otto_app/master
        7. git git checkout ext_dev_test otto_test
        8. git pull
        9. git push -u origin -o merge_request.create
